# Project Title

## Fake API

This project uses a fake API to manage plot data.

### API Endpoint

You can access the fake API at the following URL:
https://my-json-server.typicode.com/RamiroRomera/fake_api_plots/plots

### API Methods

- **GET All Plots**:  
  `GET /plots`

- **GET Single Plot**:  
  `GET /plots/:id`

- **CREATE New Plot**:  
  `POST /plots`

- **UPDATE Existing Plot**:  
  `PUT /plots/:id`

- **DELETE Plot**:  
  `DELETE /plots/:id`
